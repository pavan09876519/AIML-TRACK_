# 🌿 OptiCrop – Smart agricultural production optimization engine 

## 📖 Overview

OptiCrop is a web-based Artificial Intelligence application that helps farmers and agricultural professionals identify the most suitable crop for cultivation. By analyzing soil nutrients and environmental conditions, the system predicts the best crop using Machine Learning algorithms, enabling smarter and more productive farming decisions.

The application is built with Flask and provides an easy-to-use interface for entering agricultural parameters and receiving instant crop recommendations.

---

## 🎯 Objectives

- Recommend the best crop based on soil composition.
- Improve agricultural productivity through AI.
- Reduce incorrect crop selection.
- Support sustainable and data-driven farming.
- Provide quick and accurate predictions through a web application.

## 📊 Dataset Information

The project utilizes a crop recommendation dataset containing agricultural observations collected under different environmental conditions.

**Dataset Details**

- Total Samples: **2200**
- Crop Categories: **22**
- Prediction Type: **Multi-Class Classification**

### Input Parameters

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

### Output

- Recommended Crop
